# ls-clone
A simplified clone of the "ls" linux command in C.

<br />

Compile  
```bash
gcc ls_clone.c -o ls_clone
```

<br />

Run program  
```bash
./ls_clone
```

<br />

Options
- `-l` - Long listing (includes last modification time and file sizes).
- `-h` - Prints human-readable file sizes. To be used with `l` as in `-lh`
